# Proyecto Final Frontend - Cesar M.

Este proyecto corresponde al Proyecto Final del curso de Frontend.  
Fue desarrollado utilizando **Vue.js** y **Bootstrap**.

## Funcionalidades
- Cálculo de calificaciones con validaciones flotantes.
- Formulario de registro con validaciones de nombre, correo y contraseñas.
- Navegación por pestañas entre módulos.

## Tecnologías utilizadas
- Vue.js 3
- Bootstrap 5
- HTML5 / CSS3
